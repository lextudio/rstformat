# Security Policy

## Supported Versions

We provide security updates for each version until its End-of-Life (EOL) date. Below is a summary of currently supported versions:

| Version | Latest Release | End of Life | Notes |
| ------- | -------------- | ----------- | ----- |
| 0.2     | 0.2.0          | TBD         | Active development |
| 0.1     | 0.1.0          | 2026-06-30  | Sunset after 3 months |

## Reporting a Vulnerability

If you discover a security vulnerability in rstformat, please email
[security@lextudio.com](mailto:security@lextudio.com) with:

- A description of the vulnerability
- Steps to reproduce (if applicable)
- The affected versions
- Your suggested fix (if you have one)

Please do **not** open a public GitHub issue for security vulnerabilities.

We will acknowledge your report within 48 hours and work with you to develop
and release a patch. Thank you for helping keep rstformat secure.
